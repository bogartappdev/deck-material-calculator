# Deck Material Calculator

A static, mobile-friendly deck material estimator built with plain HTML, CSS, and JavaScript.

## Features

- Estimates deck boards with configurable board width, gap, stock length, and waste
- Estimates field joists based on on-center spacing
- Estimates rim boards, beam stock, posts, and deck screws
- Calculates a rough subtotal using editable unit prices
- Downloads a plain-text shopping list
- Runs entirely in the browser with no server or database

## Run locally

Open the folder in Visual Studio Code and either:

1. Double-click `index.html`, or
2. Install the **Live Server** VS Code extension and choose **Open with Live Server**

## Publish with GitHub Pages

1. Create a new public GitHub repository, such as `deck-material-calculator`.
2. Upload or push all files in this folder.
3. In the repository, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose the `main` branch and `/ (root)`, then save.
6. GitHub will show the public site address after deployment.

## Important limitation

This is a preliminary material estimator, not a structural engineering tool. Structural spans, loads, footings, ledger attachment, hardware, railings, stairs, and permits must comply with local code and manufacturer requirements.
